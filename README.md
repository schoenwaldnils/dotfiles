# Nils' dotfiles

## How to get started

```
# install stuff and link config files
~/path/to/dotfiles/dotfiles install

# back up config files
~/path/to/dotfiles/dotfiles backup
```

## What's included

- brew dependencies
- iTerm2 config
- npm config and dependencies
- shell functions
- zsh config and aliases
